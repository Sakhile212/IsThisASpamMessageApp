# IsThisASpamMessageApp
Spam Message Checker is a simple web application that uses regular expressions (RegEx) to detect whether a message is likely to be spam. Users can enter a message and instantly see whether it contains common spam patterns.

## Features
- Check custom messages for spam.
- Detects common spam keywords and patterns using RegEx.
- Instant feedback showing spam or non-spam results.
- Clean and user-friendly interface.

## Technologies Used
- **HTML** – Application structure.
- **CSS** – Styling and layout.
- **JavaScript** – Spam detection logic and DOM manipulation using RegEx.

## Usage
1. Open `index.html` in a web browser.
2. Enter a message in the text area.
3. Click **Check Message**.
4. View the result indicating whether the message is spam.
